🥗 Calorie & Diet Plan Calculator

A modern, interactive Calorie Calculator Web App built using HTML, CSS, and JavaScript that helps users calculate their daily calorie needs and plan their diet based on their fitness goals.

🚀 Features

🧮 Uses Mifflin–St Jeor Equation for accurate BMR

🧠 How It Works
1️⃣ BMR Calculation

Male:

BMR = 10 × weight + 6.25 × height − 5 × age + 5

Female:

BMR = 10 × weight + 6.25 × height − 5 × age − 161
2️⃣ Activity Multiplier
Activity Level	Multiplier
Sedentary	1.2
Light	1.375
Moderate	1.55
Active	1.725
3️⃣ Maintenance Calories
Maintenance Calories = BMR × Activity Multiplier
